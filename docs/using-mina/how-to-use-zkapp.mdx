# How to Use a zkApp 如何使用zkApp

<Subhead>Learn how to interact with a zero knowledge smart contract 学习如何和一个零知识智能合约交互</Subhead>


### Prerequisites 先决条件

1. Install a zkApp-compatible [Mina wallet](../using-mina/install-a-wallet). 安装一个兼容zkApp的 [Mina wallet](../using-mina/install-a-wallet)
2. Make sure that your Mina wallet contains MINA to pay for transaction fees. 确保你的Mina钱包有MINA支付交易费用。

:::note

The Mina community has created a variety of different wallets. Only the [Auro Wallet for Chrome](https://www.aurowallet.com) supports interactions with zkApps currently.	

:::

### Instructions 指令

1. Visit the zkApp in a web browser. For example, `mycoolzkapp.com`. 在web浏览器中访问zkApp。例如，`mycoolzkapp.com`.
2. Interact with the zkApp as intended. For example, make a move in a game, enter in your age, and so on. 安装计划地和zkApp交互，如在游戏中移动一下，或者输入你的年龄等等。
3. Click the confirmation button to send the transaction to the Mina network.点击确认按钮以发送交易到Mina网络中。
4. In your Mina browser wallet extension, confirm the transaction.在你的浏览器钱包插件中，确认这笔交易。
5. Done! Congratulations. Your transaction will be processed by the Mina network and, when accepted into a block, will update the zkApp's on-chain state. 完成！恭喜。你的交易将会被Mina网络处理。当它被包含入一个区块时，会更新zkApp的链上状态。


### Check the zkApp transaction 检查zkApp交易

To confirm that your zkApp transaction has been successfully processed, view the clickable transaction hash shown by the wallet on a block explorer.
你可以在区块链浏览器中查看显示在钱包里的可点击的交易hash，以确认你的zkApp交易已经被成功处理。
