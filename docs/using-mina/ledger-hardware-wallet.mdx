# Ledger Hardware Wallet Ledger 硬件钱包

Ledger has added support for Mina to their Nano S and Nano X hardware wallets. Install the Mina application on your hardware device to store your funds and interact with the Mina blockchain.
Ledger已经在它们的Nano S和Nano X硬件钱包里添加了对Mina的支持。 安装MINA应用程式到你的硬件设备，就可以存储你的资金并与Mina区块链交互。

:::info
Please note that it's not yet possible to sign zkApp transactions using a Ledger wallet.
请注意当前还未可以通过Ledger钱包来签发zkApp交易.
:::

The Mina app supports the following operations: Mina应用程式支持下面的操作:
- Generating key pairs 创建密钥对
- Signing payment transactions 签名付款交易
- Signing delegation transaction 签名委托交易


The Mina app does **not** support the following operations: Mina应用程式**不**支持下面操作：

- Signing zkApp transactions 签发zkApp交易
- Native LedgerLive wallet integrations (instead, you can use third-party wallets) 原生LedgerLive钱包集成(相反，你可以使用第三方钱包)

You can download the Mina app to your Ledger Nano S or Nano X using the instructions below. 
你可以按照下面的指令来下载Mina应用程式到你的Ledger Nano S 或者Nano X.

## Installing the Mina app 安装Mina应用程式

Before you get started, download [Ledger Live](https://www.ledger.com/ledger-live/download) if you have not done so already:
在你开始之前，下载 [Ledger Live](https://www.ledger.com/ledger-live/download):

Make sure your Ledger firmware is on the latest version supporting Mina. 
确保你的Ledger固件是支持Mina的最新版本。

1. Connect and unlock your Ledger device. 连接和解锁你的Ledger设备.
2. Open the **Manager** in [Ledger Live](https://support.ledger.com/hc/en-us/articles/4404382258961?docs=true). 在[Ledger Live](https://support.ledger.com/hc/en-us/articles/4404382258961?docs=true)里打开**Manager**
3. Allow the manager on your device. 在你的设备上允许Manager.
4. Search for **Mina** in the app catalog. 在app目录中搜索**Mina**
5. Click the **Install** button. Your device displays **Processing...**. 点击**Install** 按钮, 你的设备就会显示 **Processing...**
6. After the download is completed, Ledger Live displays **Installed**. 下载完后，Ledger Live 显示**Installed**.

## Troubleshooting 故障排除

**Why does signing a transaction take so long? 为什么签名一笔交易要如此久?** <br />
Since Mina uses new cryptography and Ledger does not have hardware acceleration support, you may experience that signing with the Mina app takes longer than other wallets. We hope that this cryptography is supported by Ledger in the future.
由于Mina使用了新的加密技术，而Ledger没有硬件加速支持，您可能会体验到使用Mina应用程序签名比其他钱包需要更长时间。我们希望这种加密技术在未来能够得到Ledger的支持。

**Why don't I see the option to update my firmware? 为何我看不到更新我的固件的选项？** <br />
If your Nano X firmware does not offer an option to upgrade to the latest version, it means your device is in the process of getting staged for the update. Just wait 1 or 2 days to see if the option is available. If not, reach out to Ledger support.
如果你的Nano X固件不提供更新到最新版本的选项，这意味着你的设备正处于准备更新的过程中。你只需再等1或2天，看看这个选项是否可用。如果依然不可用，那么就去找Ledger Support来解决。
