# How to Delegate 如何委托

<Subhead>
Learn how to delegate MINA and receive staking rewards 了解怎么委托MINA并接收staking奖励
</Subhead>

Instead of running your own node, you can delegate MINA to a validator (block producer) and receive staking rewards.
除了运行你自己的节点，你可以将MINA委托给一个验证者（区块生产者）并接收staking奖励。

### Prerequisites 先决条件

First, ensure you installed a Mina wallet with funds. 首先，确保你安装了带有资金的Mina钱包

### Delegating MINA 委托MINA

Here’s how to delegate MINA within common wallets: 下面是如何在钱包中委托MINA:

- [**Auro**](https://www.aurowallet.com) - Open the wallet, tap “*Staking*”, tap “*Go to Staking*”, select a block producer, and confirm the transaction details.

:::note

When sending transactions on a blockchain, such as Mina, senders must include a transaction fee. Most wallets calculate a recommended fee. You can also view a suggested fee amount based on current network transaction volume at this community-created [Gas Station website](https://fees.mina.tools/).

:::
