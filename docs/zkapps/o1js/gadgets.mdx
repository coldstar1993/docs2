# Gadgets

Gadgets are small, reusable low-level building blocks that simplify the process of creating new cryptographic primitives. Most gadgets build upon custom gates and act as low-level accelerators in the proof system.Gadgets是小型的、可重用的底层构建块，它简化了创建新的加密原语的过程。大多数小工具建立在自定义门上，并在证明系统中充当低级加速器。

In o1js, you can import these provable and helper methods from the `Gadgets` namespace:在o1js中，你可以从`Gadgets`命名空间中导入这些可证明的辅助方法:

- [Bitwise Operations](/zkapps/o1js/bitwise-operations)
- [Foreign Field Arithmetic](/zkapps/o1js/ecdsa)

See the type declaration for [Gadgets](/zkapps/o1js-reference/variables/Gadgets) in the o1js Reference documentation.
