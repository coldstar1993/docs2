### What are zkApps?

zkApps (zero knowledge apps) are Mina Protocol smart contracts powered by zero knowledge proofs, specifically using zk-SNARKs. zkApps(零知识应用程序)是由零知识证明驱动的Mina协议智能合约，特别是使用zk-SNARKs。

zkApps use an **off-chain execution** and mostly **off-chain state** model. This architecture allows for private computation and state that can be either private or public. zkApps使用**链下执行**和(大多数情况下)**链下状态**模型。这种架构使私有计算和可私有可公共的状态成为可能。

zkApps can perform arbitrarily-complex computations off-chain while incurring only a flat fee to send the resulting zero knowledge proof to the chain for verification of this computation. This cost saving benefit is in contrast to other blockchains that run computations on-chain and use a variable gas-fee based model. zkApps可以在链下执行任意复杂的计算，而只需要支付固定的费用就可以将生成的零知识证明发送到链上进行此计算的验证。这种节省成本的优势与其他区块链形成对比，后者在链上运行计算，并使用基于可变燃气费用的模型。

### TypeScript

zkApps are written in [TypeScript](https://www.typescriptlang.org/). zkApps是用[TypeScript](https://www.typescriptlang.org/)编写的。

TypeScript provides an easy, familiar language (JavaScript), but with type safety, making it easy to get started writing zkApps. If you're new to using TypeScript, check out this helpful 12-min introductory video [TypeScript - The Basics](https://www.youtube.com/watch?v=ahCwqrYpIuM). TypeScript提供了一种简单、熟悉的语言(JavaScript)，但具有类型安全，使您可以轻松开始编写zkApps。如果你刚开始使用TypeScript，请查看这个有用的12分钟介绍视频[TypeScript -基础知识](https://www.youtube.com/watch?v=ahCwqrYpIuM)。
