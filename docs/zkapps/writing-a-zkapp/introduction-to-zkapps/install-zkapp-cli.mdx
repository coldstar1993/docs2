# zkApp CLI Installation

Install and use the zkApp CLI to scaffold, write, test, and deploy zkApps (zero knowledge apps) for Mina Protocol using recommended best practices. 使用推荐的最佳实践来安装和使用zkApp CLI来为Mina协议搭建、编写、测试和部署zkApps(零知识应用程序)。

:::tip
To build zero knowledge apps that use [o1js](/zkapps/o1js), you only need to install the zkApp CLI. o1js is automatically included when you create a project using the zkApp CLI. In the root of your project directory, use `zk system` to show the system information with installed versions of zkApp CLI and o1js. 要构建使用[o1js](/zkapps/o1js)的零知识应用程序，你只需要安装zkApp CLI。当你使用zkApp CLI创建项目时，o1js会自动包含进来。在项目的根目录下，使用`zk system`来显示安装的zkApp CLI和o1js的系统信息。
:::

### Dependencies

- NodeJS v18 and later
- NPM v10 and later
- git v2 and later

If you have a later version of a dependency, install the required version using the package manager for your system:

- MacOs [Homebrew](https://brew.sh/)

- Windows [Chocolatey](https://chocolatey.org/)

- Linux (apt, yum, and others)

    As recommended by the Node.js project, you might need to install a recent Node.js version using NodeSource binary distributions: [Debian](https://github.com/nodesource/distributions#debinstall), [rpm](https://github.com/nodesource/distributions#rpminstall).

To verify your installed versions of dependencies, use `node -v`, `npm -v`, and `git -v`.

## Usage

To see usage information for all of the zkApp CLI commands:

```sh
$ zk --help
```

## Install the zkApp CLI

To install the latest version:

```sh
npm install -g zkapp-cli
```

To confirm successful installation:

```sh
$ zk --version
```

## Update the zkApp CLI

You are prompted to install the new version if you are running an earlier zkApp CLI minor version. For example, if you are running version 0.12.1, but the current version is 0.13.0, you are prompted to update. 如果您运行的是较早的zkApp CLI次要版本，则会提示您安装新版本。例如，如果运行的是0.12.1版本，但当前版本是0.13.0，则会提示更新。

You are not prompted to update if you are using an earlier patch version. For example, you are not notified to upgrade when you are running 0.12.0, and the current version is 0.13.1. 如果您使用的是较早的补丁版本，则不会提示您进行更新。例如，当运行0.12.0时，用户不会收到升级通知，而当前版本是0.13.1。

To update to the latest version of the zkApp CLI: 要升级到最新版本的zkApp CLI:

```sh
npm update -g zkapp-cli
```
